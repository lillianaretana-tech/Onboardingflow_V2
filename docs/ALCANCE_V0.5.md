# OnboardFlow v0.5 — Alcance incorporado

## Módulo 0: gestión de candidatos
- Validación de elegibilidad antes de enviar a RH.
- Prevención de duplicados.
- Estados: contratable, revisión RH, no contratable.
- Sin exposición de motivos al supervisor.
- Registro del candidato y envío a bandeja de RH.

## Recursos Humanos
- Bandeja de nuevos candidatos.
- Aprobación, devolución o rechazo.
- Generación de código.
- Copia de invitación.
- Exportación compatible con Excel.
- Gestión confidencial de elegibilidad.
- Control de inducción.
- Constructor de empresas.

## Candidato
- Nombre + cédula + código.
- Seis intentos.
- Reglas.
- Prueba de equipo.
- Tutorial de Teams.
- Sala de espera.
- Módulos.

## Seguridad
- Acceso administrativo separado.
- RLS por tenant.
- Observaciones internas solo para RH.
- No existe selector público de roles.
